### SQL 脚本

| 文件 | 说明 | 用途 |
|------|------|------|
| [DATABASE_SETUP.sql](./sql/DATABASE_SETUP.sql) | 完整数据库脚本 | 全新部署 |

**包含内容**：
- ✅ 所有表结构（17 张表）
- ✅ 索引优化
- ✅ RLS 安全策略
- ✅ 表注释

### 数据库架构

```
核心表：
─── schedules              # 个人日程（支持日期范围、优先级、提醒）
─── schedule_updates       # 更新记录
─── schedule_attachments   # 日程附件
─── schedule_tags          # 日程标签（新增v2.5）
─── teams                  # 团队
─── team_members           # 成员
─── work_groups            # 工作组
─── tasks                  # 任务
─── work_items             # WBS 子项
─── task_comments          # 评论
─── task_attachments       # 任务附件（新增v2.5）
─── task_dependencies      # 依赖关系（新增v2.5）
─── notifications          # 通知系统（新增v2.5）
─── audit_logs             # 审计日志（新增v2.5）
└── team_invitations       # 邀请
```

## 🎯 使用场景

### 个人使用

1. **项目开发追踪**
   - 创建日程：开发某功能
   - 定期更新：记录开发进度
   - 附件：代码截图、测试报告

2. **学习计划管理**
   - 创建日程：学习某技术
   - 阶段更新：学习笔记、心得
   - 附件：学习资料、练习代码

3. **论文写作**
   - 创建日程：论文写作计划
   - 更新记录：各章节完成情况
   - 附件：参考文献、数据图表

### 团队使用

1. **产品开发**
   - 团队：产品研发团队
   - 工作组：前端组、后端组
   - 任务：功能模块
   - WBS：具体开发任务

2. **科研项目**
   - 团队：科研课题组
   - 工作组：实验组、分析组
   - 任务：实验计划
   - 附件：实验数据、论文
