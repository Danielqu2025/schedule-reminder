# 项目更新日志 (Changelog)

## v2.3.0 (2026-01-31)
- **文件结构优化**：大幅简化项目文件，合并冗余的文档和 SQL 脚本。
- **文档重构**：将零散的更新说明合并为统一的 `CHANGELOG.md`，开发指南整合为 `DEVELOPMENT_GUIDE.md`。
- **数据库脚本简化**：整合所有数据库初始化逻辑至 `FULL_DATABASE_SETUP.sql`，提供一键式部署方案。

## v2.2.0 (2026-01-28)
- **RLS 架构重构**：引入 `SECURITY DEFINER` 安全辅助函数，彻底解决无限递归问题。
- **查询性能优化**：针对提醒引擎和任务筛选增加了 6 个核心数据库索引。
- **代码重构**：将团队邀请逻辑提取为独立的 `TeamInvitationManager` 组件。
- **提醒逻辑改进**：优化了 `useReminderEngine` 的查询效率。

## v2.1.0
- **团队邀请系统**：支持通过邮箱发送团队加入邀请。
- **工作组管理**：支持将团队成员划分为不同的工作组。
- **任务 WBS 拆分**：支持将任务拆分为多个并行或顺序执行的工作子项。

## v1.0.0
- **基础功能**：实现个人日程管理、登录注册及基础提醒功能。
